# HR-Analytics-Dashboard-Power-BI

This project analyzes employee attrition trends using Power BI.

## Key Insights

- Total Employees: 1470
- Attrition Count: 237
- Attrition Rate: 16.1%
- Average Age: 37
- Average Salary: 6.5K
- Average Years at Company: 7

## Dashboard Features

- Attrition analysis by Age Group
- Attrition analysis by Salary Slab
- Attrition analysis by Education Field
- Attrition analysis by Job Role
- Department-level filtering using slicers
- Interactive KPI cards

## Tools Used

Power BI  
Power Query  
Data Cleaning
Data Visualization  


## Objective

Identify key factors affecting employee attrition and support HR decision-making using interactive dashboards.
